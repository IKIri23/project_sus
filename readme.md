# Projeto 04 - Sistema Web de Gestão de Saúde (SUS Digital)

Este projeto é um webservice desenvolvido em **Node.js** com **Express** e **EJS**, criado para compor a nota da P2. O sistema simula um painel de gestão para a saúde pública (SUS), apresentando serviços preventivos, novas incorporações e auditoria de unidades.

## Tecnologias Utilizadas
**Node.js**: Ambiente de execução.
**Express.js**: Framework para criação das rotas e do servidor.
**EJS**: Motor de visualização para renderizar páginas HTML dinâmicas.
**JSON**: Banco de dados estático para consumo das rotas.

## Pré-requisitos
Para rodar este projeto localmente, você precisará ter o [Node.js](https://nodejs.org/) instalado em sua máquina.
